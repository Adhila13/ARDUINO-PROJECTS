# Arduino Seven Segment Display

## Description
This project demonstrates how to interface a seven-segment display with an Arduino Uno.

The display counts from 9 to 0 by controlling the seven segments (A-G) individually.

## Components Used
- Arduino Uno
- Seven Segment Display
- Resistors
- Breadboard
- Jumper Wires

## Concepts Learned
- Digital Output Pins
- Seven Segment Display Operation
- Common Cathode Display
- Arduino Programming
- Sequential Display Control

## Output
The display automatically counts from 9 down to 0 with a delay between each number.
