# Push Button Controlled LED 

## Objective
Control an LED using a push button.

## Components Used
- Arduino Uno
- Push Button
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

## Concepts Used
- Digital Input
- Digital Output
- Conditional Statements
- Push Button Interface

## Working
When the button is pressed, the LED changes state according to the program logic.

## Learning Outcome
Learned how to read digital inputs from a push button and control an output device (LED) using Arduino.
