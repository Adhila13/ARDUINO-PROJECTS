# Traffic Light Simulation

Arduino traffic light simulation using three LEDs.

## Components Used
- Arduino Uno
- Red LED
- Yellow LED
- Green LED
- 3 Resistors
- Breadboard
- Jumper Wires

## Working
The traffic light follows the sequence:

Red → Red + Yellow → Green → Yellow → Repeat

## Concepts Learned
- Digital Output
- Multiple LED Control
- Delay Function
- Sequential Logic
